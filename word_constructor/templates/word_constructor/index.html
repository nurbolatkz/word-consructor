Загрузите .docx, при необходимости добавьте список параметров 1С. Документ откроется в OnlyOffice, где можно редактировать файл и отправить результат обратно.

Заполнить шаблон и открыть редактор

Этот режим использует шаблон с плейсхолдерами {{ключ}} или [Ключ] и значения параметров.

API для 1С

POST/services/word-constructor/api/1c/replace

Без сессии: отправить .docx и JSON {"Ключ":"Значение"}, получить готовый .docx в ответ.

POST/services/word-constructor/api/1c/replace-edit

Создать ручную OnlyOffice-сессию из замененного документа. 1С получает editor_url для пользователя и update_url для кнопки Обновить.

POST/services/word-constructor/api/1c/template-builder/bridge

1С отправляет Word и список параметров, получает session_id, builder_url, status_url, download_url.

GET/services/word-constructor/api/template-builder/{id}/status

Опрос до status = ready.

GET/services/word-constructor/api/template-builder/{id}/download

Скачать готовый отредактированный .docx.
