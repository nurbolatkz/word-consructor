Word

Конструктор приказов

Сервис принимает шаблон Word-документа (.docx) с плейсхолдерами вида {{ключ}} и параметры из 1С, автоматически заполняет документ и открывает редактор, где пользователь может изменить значения перетаскиванием или вручную, а затем скачать готовый файл.

API для 1С

POST /services/word-constructor/api/create

Создать сессию: передать шаблон и начальные параметры. Тип запроса: multipart/form-data

  • template — файл .docx с плейсхолдерами {{ключ}}
  • params — JSON-строка: {"ФИО": "Алексей Иванов", "Должность": "Менеджер"}
{
  "id": "a3f8c...",
  "editor_url": "/services/word-constructor/editor/a3f8c...",
  "expires_at": "2026-04-10T14:00:00+00:00"
}
PATCH / POST /services/word-constructor/api/sessions/{id}/params

Добавить или обновить параметры в открытой сессии — документ в редакторе не перерисовывается, обновляется только панель параметров. Тело: application/json 

POST /api/sessions/a3f8c.../params
Content-Type: application/json

{"Приказ №": "47-К", "Дата": "10.04.2026"}
{
  "params_version": 2,
  "params": { "ФИО": "Алексей Иванов", "Приказ №": "47-К", "Дата": "10.04.2026" }
}
