KazUni Google Docs Test

Это минимальная тестовая страница. Она не интегрирует Google Docs в шаблонизатор, а просто открывает новый документ Google Docs как верхнеуровневую страницу, чтобы проверить, способен ли встроенный браузер 1С отрисовать редактор Google Docs.

Если Google Docs откроется и будет работать в 1С, можно будет строить следующую интеграцию вокруг него. Если страница зависнет или останется пустой, значит встроенный движок 1С для этого сценария не подходит.
